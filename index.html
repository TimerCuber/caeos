CAE OS

Un sistema operativo ligero, rápido y académico

Introducción

CAE OS surge como un proyecto académico enfocado en la creación de un sistema operativo liviano y funcional, diseñado para equipos de bajos recursos. Su propósito es brindar a los estudiantes y usuarios en general una herramienta optimizada que permita aprovechar al máximo el hardware disponible.

Estado del arte

Actualmente existen múltiples sistemas operativos ligeros basados en Linux, como Lubuntu, Xubuntu y Linux Mint XFCE. Sin embargo, estos sistemas no siempre están adaptados al contexto educativo y académico local. CAE OS toma como base las mejores características de estas distribuciones, pero se centra en la personalización y facilidad de uso para estudiantes.

Metodología

La metodología empleada para el desarrollo de CAE OS se basa en los siguientes pasos:

  • Selección de la distribución base: Debian 12 con XFCE.
  • Optimización del sistema para equipos de bajos recursos.
  • Configuración de aplicaciones esenciales para el entorno académico.
  • Creación de una ISO personalizada con Refracta Snapshot.
  • Pruebas en diferentes equipos para evaluar compatibilidad y rendimiento.

Resultados

Las pruebas iniciales de CAE OS muestran un arranque rápido, bajo consumo de memoria y una interfaz intuitiva. Se ha logrado ejecutar en equipos con procesadores antiguos y poca memoria RAM, manteniendo un rendimiento fluido para tareas básicas de oficina, navegación web y uso académico.

Conclusión

CAE OS representa una alternativa eficiente y accesible frente a otros sistemas operativos, aportando una solución práctica para la reutilización de computadores en entornos académicos. A futuro, se buscará implementar un instalador automático y una mayor integración con aplicaciones educativas.
